# NoteForge 产品说明书

**版本：0.1.0 | 更新日期：2026年07月**

---

## 1. 产品概述

NoteForge 是一款纯 Rust 实现的本地优先（local-first）Markdown 知识管理桌面应用，对标 Obsidian。

### 设计哲学

1. **数据主权**：文件是唯一真实数据源
2. **性能**：启动快、大 Vault 不卡顿
3. **可扩展**：WASM 插件体系
4. **安全**：插件默认沙箱化

## 2. 系统架构

## 3. 功能列表

### 桌面应用
- **打开 Vault**：目录选择对话框打开笔记库
- **文件树**：左侧侧栏显示笔记列表
- **四种编辑模式**：源码 / 分栏 / 即输即显 / 预览
- **Markdown 渲染**：comrak 引擎（CommonMark + GFM）
- **代码高亮**：highlight.js（190+ 语言）
- **Wikilink 导航**：`[[链接]]` 点击跳转
- **实时预览**：源码修改 150ms 内自动刷新预览
- **Typora 模式**：contentEditable 直接编辑渲染结果
- **快速切换器** Ctrl+O：搜索并打开文件
- **命令面板** Ctrl+P：搜索并执行命令
- **全文搜索**：搜索笔记内容
- **反链显示**：引用当前笔记的链接列表
- **新建笔记**：创建空 .md 文件
- **保存** Ctrl+S：写回文件
- **Vault 统计**：笔记数、附件数、目录数、大小
- **菜单栏**：文件/编辑/视图/工具/帮助（含下拉菜单）

### CLI 工具
```bash
nf-vaultgen generate --profile smoke --seed 42 --out ./vault
nf-vaultgen list-profiles
nf-app info ./vault/vault
```
