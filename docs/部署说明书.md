# NoteForge 部署说明书

**版本：0.1.0 | 更新日期：2026年07月**

---

## 1. 系统要求

- **Rust**：1.70+（stable）
- **Node.js**：18+（推荐 20+）
- **系统库**（Linux）：`libxkbcommon fontconfig libgl`

## 2. 安装

```bash
# 安装 Tauri CLI
cargo install tauri-cli --version "^2"

# 安装前端依赖
cd frontend && npm install

# 构建并运行（开发模式）
cargo tauri dev

# 构建发布版本
npm run build && cargo build -p noteforge --release
```

## 3. Arch Linux / Manjaro 打包

```bash
cd pkg/manjaro
makepkg -si
```

## 4. 运行

```bash
# 桌面应用
cargo tauri dev

# 生成测试库
cargo run -p nf-vaultgen -- generate --profile smoke --seed 42 --out ./vault

# CLI 浏览
cargo run -p nf-app -- info ./vault/vault
```

## 5. 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+O | 快速切换器 |
| Ctrl+P | 命令面板 |
| Ctrl+S | 保存当前笔记 |

## 6. 项目结构

```
noteforge/
├── src-tauri/          # Tauri 桌面应用 (Rust)
│   ├── src/main.rs     # IPC 命令
│   └── tauri.conf.json
├── frontend/           # React + TypeScript 前端
│   ├── src/App.tsx     # 主应用
│   ├── src/components/ # UI 组件
│   └── src/layout/     # 布局状态机
├── crates/             # Rust crate 库
│   ├── nf-core/        # 领域模型
│   ├── nf-vault/       # 文件系统
│   ├── nf-markdown/    # Markdown 解析
│   ├── nf-render/      # comrak HTML 渲染
│   ├── nf-index/       # 元数据缓存
│   ├── nf-editor/      # 编辑器引擎
│   ├── nf-graph/       # 图谱
│   ├── nf-plugin/      # 插件系统
│   ├── nf-workspace/   # 工作区持久化
│   ├── nf-vaultgen/    # 测试库生成器
│   └── nf-app/         # CLI 应用
├── docs/               # 文档
└── pkg/manjaro/        # Arch Linux 打包
```

## 7. 测试

```bash
cargo test --workspace
cd frontend && npm run build
```
